**Multi-Disease Prediction System**

A machine learning project that predicts possible diseases based on user symptoms using trained ML models.

**Features**

•	Disease prediction from symptoms 

•	Simple Streamlit interface 

•	Multiple ML models used 

•	Fast and easy to use 

**Tech Stack**

•	Python 

•	Streamlit 

•	Scikit-learn 

•	Pandas 

•	NumPy 

**Models Used**

•	Random Forest 

•	Decision Tree 

•	Naive Bayes 

**Workflow**

1.	User enters symptoms 

2.	Data is preprocessed 

3.	ML model predicts disease 

4.	Result is displayed
